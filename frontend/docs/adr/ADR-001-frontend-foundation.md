# ADR-001: Frontend Foundation with React + TypeScript + Vite

## Status
Accepted

## Context
We need to start the web phase quickly while keeping code interview-ready, maintainable, and aligned with backend API contracts.

## Decision
Use React + TypeScript with Vite for frontend foundation.

## Consequences
- Fast development feedback loop.
- Type-safe API integration with backend contracts.
- Clean base for future routing/state/testing additions.
