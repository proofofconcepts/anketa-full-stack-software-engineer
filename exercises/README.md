# Refactoring Exercises

These exercises simulate real interview tasks: identify what's wrong with the given code and refactor it to production quality.

Each exercise contains:
- `poor-code.*` — the original broken/naive implementation
- `EXERCISE.md` — what problems to identify and what the refactored result must satisfy
- `solution/` — reference solution (attempt on your own first)

## Exercises

| # | Title | Stack | Key concepts |
|---|-------|-------|--------------|
| 01 | NestJS Users Search | NestJS / TypeScript | Service layer, DTOs, Prisma, Swagger, validation |
| 02 | Flutter Search Page | Flutter / Dart | BLoC, Clean Architecture, debounce, dispose |
| 03 | React Poll Feed | React / TypeScript | Data fetching, AbortController, memoisation, optimistic update |
| 04 | NestJS Auth Module | NestJS / TypeScript | bcrypt, JWT, Passport, guards, security |
| 05 | Flutter Create Poll | Flutter / Dart | BLoC form, Clean Architecture, navigation, error handling |
| 06 | React Voting App | React / TypeScript | Zustand, TanStack Router, code splitting, performance |
